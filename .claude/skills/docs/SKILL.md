# Docs

Write or update docs.

- Follow `references/documentation-standard.md`.
- Load exactly one template when a template is needed:
  - API docs → `templates/api-doc.md` only.
  - Operational docs → `templates/runbook.md` only.
- Never load both templates in one execution.
- If the doc type is unclear, ask one short clarifying question first.
