# Leadership

- [ ] Clarity over control: define outcomes, not steps
- [ ] Decisions written down and shared
- [ ] Feedback specific, timely, kind
- [ ] Unblock before directing
- [ ] Protect focus time
- [ ] Bias to small, reversible steps
- [ ] Credit publicly, correct privately
- [ ] Optimize for team throughput, not personal output
